# 🚀 Dropeni – Full-Stack Logistics & E-commerce Platform

**Dropeni** is a full-stack logistics and e-commerce platform designed to simplify **international shipping** and **Shop & Ship services**. It enables users to seamlessly order products, manage deliveries, and track shipments from origin to destination — all within a unified digital ecosystem.

This repository contains the **backend API**, **admin dashboard**, and **mobile application** that power the Dropeni platform.

---

## 🌍 Overview

Dropeni bridges the gap between local commerce and global delivery by providing:

* 📦 **Logistics Services** – Ship items internationally with ease
* 🛒 **Shop & Ship** – Order products locally and get them delivered abroad
* 📱 **Mobile-first Experience** – Seamless booking and tracking on mobile devices
* 🧑‍💼 **Admin Control Panel** – Manage operations, users, and shipments efficiently

---

## 🏗️ Architecture

The project is structured into three core components:

### 1. Backend API

Handles all server-side logic, including:

* User authentication & authorization
* Shipment booking & tracking
* Payment processing integration
* Order management
* Notifications & system workflows

### 2. Admin Dashboard

A web-based interface for internal operations:

* Manage shipments and orders
* Monitor system activity
* Handle customer requests
* Configure services and pricing

### 3. Mobile Application

Customer-facing application that allows users to:

* Create and manage shipments
* Track deliveries in real-time
* Shop products for international delivery
* Receive notifications and updates

## 🚀 Getting Started

### Prerequisites

* Node.js installed
* Package manager (npm or yarn)
* Database setup (MongoDB)

### Installation

```bash
# Clone the repository
git clone https://github.com/obinnakelvin/dropeni.git

# Navigate into the project
cd dropeni

# Install dependencies
npm install
```

### Running the Application

```bash
# Start backend server
npm run dev

# Start admin dashboard
npm run dev

# Start mobile app
npm run mobile
```
